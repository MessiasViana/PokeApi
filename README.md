# PokeApi
Projeto que lista todos pokemons, com sistema de busca, paginação e localStorage

## Instalar node_modules

Para usuários de Yarn:
* `"yarn install"`

Para usuários de npm:
* `"npm install"`


## Rodar projeto

Para usuários de Yarn:
* `"yarn start"`

Para usuários de npm:
* `"npm run start"`
